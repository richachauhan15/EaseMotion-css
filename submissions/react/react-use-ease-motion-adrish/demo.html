useEaseMotion — Demo

Two <AnimatedCard /> instances, each driven by useEaseMotion(tokens). Open devtools and inspect either card's DOM node — you'll see the real em="..." attribute set on mount, which is what the engine's runtime.js MutationObserver picks up in the full EaseMotion CSS build.

This page has no server dependency and no build step — it loads React, ReactDOM, and Babel standalone from a CDN and compiles the JSX below in-browser. It is intentionally NOT wired to the full EaseMotion engine bundle (parser.js / compiler.js / runtime.js), since this submission adds no engine files; the point of this demo is only to show the hook correctly setting the em attribute on the DOM node it's attached to, which you can verify by inspecting the elements above.
