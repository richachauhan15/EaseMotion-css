# Ease DaemonSet Deployment Health

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes DaemonSet deployment status across cluster nodes.

## Features

- DaemonSet deployment overview
- Rolling update monitoring
- Node scheduling visibility
- Deployment failure alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable Kubernetes infrastructure component for tracking DaemonSet deployments, monitoring node coverage, and identifying rollout issues before they affect cluster-wide services.
