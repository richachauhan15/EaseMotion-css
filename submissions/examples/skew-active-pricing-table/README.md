# CSS Skew Active Pricing Table

A responsive pricing table built using pure HTML and CSS featuring a skew animation on hover and an active highlighted pricing plan.

## Features

- Pure HTML & CSS
- Responsive Design
- Skew Hover Animation
- Highlighted Active Plan
- CSS Variables
- Accessible
- prefers-reduced-motion Support
- No JavaScript Required

## Folder Structure

Simply open `demo.html` in any modern browser.

## Accessibility

Supports the `prefers-reduced-motion` media query to reduce motion for users who request it.
