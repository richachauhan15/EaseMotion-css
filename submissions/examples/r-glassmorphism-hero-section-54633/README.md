# CSS Glassmorphism Hero Section

A modern **Glassmorphism Hero Section** built using pure HTML5 and CSS3 for creative portfolio layouts. It features a frosted-glass interface, animated floating background shapes, responsive design, and accessibility support—all without JavaScript.

---

## ✨ Features

- Pure HTML5 & CSS3
- Glassmorphism (frosted glass) UI
- Animated floating gradient background
- Modern gradient CTA buttons
- Responsive hero layout
- CSS Custom Properties
- `backdrop-filter` glass effect
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Link the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in any modern browser.
