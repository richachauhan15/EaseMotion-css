How It Works

  • Pure CSS: Waveform traces use CSS keyframe animations with clip-path polygon morphing to simulate seismic waveform patterns.
  • Grid layer: Optional reference grid drawn with CSS repeating-linear-gradient for amplitude measurement.
  • Variants: Standard, compact, alert (red), dark (inverted), slow drift, and dual-channel modes.
  • Dark mode: Automatically adapts via prefers-color-scheme.
  • Accessibility: ARIA roles with meaningful labels. Respects prefers-reduced-motion by pausing all animations.
  • High contrast & forced-colors: Adapts to system accessibility preferences.
  • Zero JavaScript: All animation and state is driven by CSS alone.
