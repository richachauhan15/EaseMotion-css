# CSS 3D Flip Modal

## Overview

A lightweight pure HTML and CSS modal component with a smooth 3D flip animation effect.

This component does not require JavaScript and follows the EaseMotion CSS animation-first philosophy.

## Features

- Pure HTML & CSS
- 3D flip entrance animation
- Responsive design
- No JavaScript dependency
- Smooth CSS transitions
- Accessible toggle mechanism
- prefers-reduced-motion support
