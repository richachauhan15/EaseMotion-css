# Text Reveal Hover Utility

## Description

A reusable CSS hover utility that smoothly replaces one text with another using a vertical slide and fade animation.

## Features

- Smooth text reveal animation
- Vertical slide transition
- Pure CSS implementation
- Lightweight and reusable
- Suitable for buttons, cards, and interactive UI elements

## Usage

```html
<div class="text-reveal-hover">
    <span class="primary-text">Hover Me</span>
    <span class="secondary-text">Welcome!</span>
</div>
```
