# CSS Float Drift Popover

A responsive **Float Drift Popover** built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Floating drift animation
- Glassmorphism design
- Fully responsive
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
float-drift-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser and hover over the button to display the animated popover.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Animations are disabled automatically for users who enable **Reduce Motion**.
