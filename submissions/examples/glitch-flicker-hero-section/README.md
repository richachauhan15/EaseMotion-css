# CSS Glitch-Flicker Hero Section

A responsive hero section built using pure HTML and CSS featuring a glitch flicker text animation with accessible motion support.

## Features

- Pure HTML & CSS
- Responsive Hero Layout
- CSS Glitch Text Effect
- Flicker Animation
- CSS Variables
- Accessible
- prefers-reduced-motion Support
- No JavaScript

## Folder Structure

```
glitch-flicker-hero-section/
├── demo.html
├── style.css
└── README.md
```

## CSS Variables

```css
--primary
--secondary
--bg
--text
--transition
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports the `prefers-reduced-motion` media query to disable animations for users who prefer reduced motion.
