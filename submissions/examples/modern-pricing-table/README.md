# Modern Pricing Table

## Description

A responsive pricing table component built using only HTML and CSS. It includes multiple pricing plans, feature lists, highlighted popular plans, and call-to-action buttons.

## Features

- Responsive layout
- Three pricing cards
- Featured plan badge
- Modern shadow effects
- Hover animation
- Pure HTML & CSS
- Easy to customize

## Usage

```html
<div class="pricing-table">
    <div class="pricing-card">
        ...
    </div>

    <div class="pricing-card featured">
        ...
    </div>

    <div class="pricing-card">
        ...
    </div>
</div>
```
