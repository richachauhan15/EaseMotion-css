# Interactive Animation Playground

An interactive playground for experimenting with CSS animations in real time.

## Features

- Live animation preview
- Multiple animation types
- Adjustable duration
- Adjustable delay
- Iteration count control
- Easing selection
- Copy generated CSS
- Responsive design

## Controls

- Animation Type
- Duration
- Delay
- Iterations
- Easing Function

## Preview

The animation preview updates instantly when the user clicks the Play Animation button.
