# CSS Rotate-Fade Pricing Table

A responsive pricing table built using pure HTML and CSS featuring a smooth rotate-and-fade hover animation.

## Features

- Pure HTML & CSS
- Rotate & Fade Hover Animation
- Responsive Design
- Highlighted Featured Plan
- CSS Variables
- Accessible
- `prefers-reduced-motion` Support
- No JavaScript Required

## Folder Structure

```

├── demo.html

```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports the `prefers-reduced-motion` media query to reduce or disable animations for users who prefer reduced motion.
