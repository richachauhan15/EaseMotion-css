# CSS Scale Hover Drawer

A responsive Scale Hover Drawer built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Smooth scale-on-hover animation
- Glassmorphism-inspired design
- Fully responsive
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
scale-hover-drawer/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser.

Hover over the drawer to see the smooth scaling animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Users with **Reduce Motion** enabled will automatically receive a non-animated version.
