# CSS Rotate Fade Drawer

A responsive Rotate Fade Drawer built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Rotate + Fade entrance animation
- Glassmorphism UI
- Responsive layout
- No JavaScript
- Accessible with `prefers-reduced-motion`

## Folder Structure

```
rotate-fade-drawer/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser.

The drawer animates into view using a smooth rotate and fade effect.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Users who enable **Reduce Motion** will automatically receive a non-animated version.
