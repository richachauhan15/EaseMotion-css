# Scale Hover Dropdown

A responsive dropdown component built using pure HTML5 and CSS3 with a smooth scale-hover animation.

## Features

- Pure HTML & CSS
- Smooth scale animation
- Hover & keyboard accessibility (`:focus-within`)
- Responsive layout
- Modern portfolio design
- Supports `prefers-reduced-motion`
- No JavaScript required

## Folder Structure

```
scale-hover-dropdown/
├── demo.html
├── style.css
└── README.md
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Usage

Open `demo.html` in any modern browser.
