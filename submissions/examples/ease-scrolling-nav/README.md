# 📜 Scrolling Nav – Hides/Shows on Scroll Direction

> A smart, space-saving navigation bar that automatically hides when scrolling down and reappears when scrolling up, creating a clean, immersive browsing experience.

---

## 📖 Description

The **Scrolling Nav** is a smart navigation component that automatically hides when you scroll down and reappears when you scroll up. This creates a clean, immersive browsing experience where content takes center stage, and the navigation is always accessible when needed. The component uses smooth CSS transforms with spring easing for a buttery-smooth transition.

The navigation features a modern, glass-morphism design with a blurred background, a brand logo, navigation links, and a mobile-responsive hamburger menu. It's perfect for content-heavy websites, blogs, portfolios, and any application where maximizing screen real estate is important.
