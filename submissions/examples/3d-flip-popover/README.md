# CSS 3D Flip Popover

A responsive **3D Flip Popover** built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Smooth 3D flip animation
- Glassmorphism-inspired design
- Fully responsive
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
3d-flip-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser and hover over the button to display the 3D flip popover.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Animations are automatically disabled for users who enable **Reduce Motion**.
