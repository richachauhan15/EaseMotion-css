# 🍿 Popcorn Nav – Items Pop Out Sequentially

> A playful, energetic navigation menu where items pop out one after another like popcorn, creating a delightful, cascading reveal effect.

---

## 📖 Description

The **Popcorn Nav** is a playful, energetic navigation component where menu items pop out sequentially like popcorn kernels. Each item bursts into view with a staggered delay, scaling up from a small dot with a bouncy, spring-like motion. The center toggle button transforms from a hamburger icon to a close (X) icon when the menu is open.

The component features a circular toggle button that reveals a vertical list of navigation links. Each link has a unique color icon and includes a tooltip label on hover. The staggered delays (50ms increments) create an organic, cascading effect that feels energetic and fun. Items have a slight bounce variation, making each pop feel unique.

Perfect for playful interfaces, creative portfolios, mobile navigation, and any application where you want to add energy and personality to your menu. The Popcorn Nav transforms a simple navigation into a delightful, engaging experience.
