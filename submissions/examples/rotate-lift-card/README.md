# Rotate Lift Card Hover Utility

## Description

A reusable CSS utility that adds a smooth rotate and lift animation when a card is hovered. The effect creates a modern and engaging interaction suitable for feature cards, portfolios, dashboards, and product showcases.

## Features

- Smooth rotate animation
- Lift effect on hover
- Enhanced shadow
- Lightweight and reusable
- Pure HTML & CSS

## Usage

```html
<div class="card">
    <h2>Rotate Lift Card</h2>
    <p>Hover over this card.</p>
</div>
```
