# Ease Container Restart Analysis Board

## Overview

A responsive HTML and CSS dashboard component for monitoring container restart activity, workload stability, and CrashLoopBackOff risks.

## Features

- Restart count monitoring
- Stable workload visibility
- Restart spike detection
- Critical failure alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable Kubernetes monitoring component that helps operations teams identify unstable workloads, reduce downtime, and improve application reliability.
