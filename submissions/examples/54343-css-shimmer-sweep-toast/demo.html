How It Works

Each toast uses a checkbox toggle for dismiss. The shimmer sweep effect is a CSS pseudo-element with a gradient that slides across the toast. Progress bars auto-time using CSS animation.

Accessibility

Toasts include role="alert" and aria-live="polite" for screen readers. The prefers-reduced-motion media query disables all animations for users who prefer reduced motion.
