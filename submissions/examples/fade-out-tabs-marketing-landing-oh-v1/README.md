# 📈 Fade-Out Tabs Component Variant (Marketing Landing Page Style)

## 📋 Component Overview

A pure CSS animated **Tabs** component featuring smooth **Fade-Out** interaction transitions, styled for **Marketing Landing Page** interfaces. High-converting landing pages with features, pricing, testimonials, and FAQ sections.

## 🛠️ Folder & Structural Architecture

```
submissions/examples/fade-out-tabs-marketing-landing-oh/
├── demo.html     # Live preview with marketing landing design
├── style.css     # Marketing Landing fade-out tab animation styles
└── README.md     # Production integration documentation
```

## 🔧 CSS Custom Properties

| Property | Default | Description |
|----------|---------|-------------|
| `--tab-transition-duration` | `0.4s` | Animation transition duration |
| `--tab-transition-ease` | `cubic-bezier(0.4, 0, 0.2, 1)` | Easing function |
| `--tab-fade-opacity` | `0` | Opacity for inactive tabs |

## 🎯 Animation Variants

### 1. **Fade** - Simple opacity fade
### 2. **Slide** - Fade with vertical slide
### 3. **Pop** - Fade with bounce effect

## 🎨 Visual Features

1. **Features Grid**: Icon-based feature cards
2. **Pricing Tables**: Three-tier pricing with featured highlight
3. **Testimonials**: Customer review cards
4. **FAQ Section**: Accordion-style questions
5. **Gradient Accents**: Orange to purple gradients
6. **Manrope Typography**: Modern marketing font
