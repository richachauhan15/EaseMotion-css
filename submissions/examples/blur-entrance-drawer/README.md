# Blur Entrance Drawer

A responsive CSS blur entrance drawer animation for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Blur entrance animation
- Glassmorphism UI
- Responsive design
- Accessible
- Supports `prefers-reduced-motion`

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in any browser.

## Animation

The drawer enters from the left with:

- Blur effect
- Fade in
- Slide animation
- Smooth easing

## Responsive

Works on:

- Desktop
- Tablet
- Mobile

## Accessibility

Users who prefer reduced motion automatically receive a non-animated version.
