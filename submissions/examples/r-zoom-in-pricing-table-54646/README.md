# CSS Zoom-In Pricing Table

A responsive **Zoom-In Pricing Table** built using only HTML and CSS for creative portfolio layouts.

---

## Features

- Pure HTML5 & CSS3
- Smooth Zoom-In entrance animation
- Hover zoom interaction
- Featured pricing plan
- Responsive layout
- CSS Custom Properties
- Accessible design
- Supports `prefers-reduced-motion`
- No JavaScript required

## Usage

Include the stylesheet.

```html
<link rel="stylesheet" href="style.css">
```

Copy the pricing table markup from `demo.html` into your project.
