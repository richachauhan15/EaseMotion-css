# Ease Horizontal Pod Autoscaler Status

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes Horizontal Pod Autoscaler (HPA) activity and scaling health.

## Features

- Autoscaler health overview
- Active scaling monitoring
- Resource utilization alerts
- Scaling failure detection
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable Kubernetes monitoring component for visualizing autoscaling behavior, tracking workload growth, and identifying scaling bottlenecks before they impact production services.
