# Ease Pod Eviction Insights Panel

## Overview

A responsive HTML and CSS component for monitoring Kubernetes pod eviction events and workload stability.

## Features

- Pod eviction monitoring
- Resource pressure visibility
- Recovery status tracking
- Critical eviction alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable Kubernetes operations component for tracking pod eviction events, identifying resource pressure, and improving cluster reliability through clear operational visibility.
