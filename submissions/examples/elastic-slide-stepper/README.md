# Elastic Slide Stepper

A lightweight pure CSS stepper component with elastic sliding animations.

## Features

- Pure HTML and CSS
- No JavaScript dependency
- Smooth elastic slide animation
- Responsive design
- Accessible reduced motion support

## Usage

```html
<div class="step">
  <div class="circle">1</div>
  <div class="content">
    <h3>Step Title</h3>
  </div>
</div>
