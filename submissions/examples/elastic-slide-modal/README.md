# CSS Elastic Slide Modal

## Overview

A lightweight responsive modal built entirely with HTML and CSS. The modal opens with an elastic slide animation and requires no JavaScript.

## Features

- Pure HTML & CSS
- Elastic slide animation
- Responsive design
- Accessible checkbox toggle
- No JavaScript
- prefers-reduced-motion support
- Easy customization

## Folder Structure

```
elastic-slide-modal/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<input type="checkbox" id="modal-toggle">

<label for="modal-toggle" class="open-btn">
Open Modal
</label>

<div class="modal">
...
</div>
```

## Customization

You can customize:

- Background color
- Animation timing
- Border radius
- Shadow
- Button colors
