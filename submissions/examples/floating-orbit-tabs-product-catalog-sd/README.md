# Floating Orbit Tabs

## What does this do?

A CSS-only tab component with a subtle floating orbit animation on the active tab. The active tab gently moves in a small circular path while scaling up, creating a floating effect.

## How is it used?

Include `style.css` and use the `.floating-orbit-tabs` class with radio inputs, labels, and panel divs. Customize via CSS custom properties: `--tabs-duration`, `--tabs-easing`, `--tabs-orbit-distance`, `--tabs-accent-color`, `--tabs-scale`.

## Why is it useful?

Provides a lightweight, accessible tab component with a distinctive floating interaction that draws attention to the active tab without JavaScript.
