# Rotate Fade Pricing Table

A modern creative portfolio pricing table built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Rotate + Fade entrance animation
- Glassmorphism cards
- Featured pricing plan
- Hover lift and rotation effect
- Fully responsive
- Supports `prefers-reduced-motion`
- No JavaScript required
