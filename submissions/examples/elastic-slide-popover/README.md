# CSS Elastic Slide Popover

A responsive **Elastic Slide Popover** built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Elastic slide entrance animation
- Glassmorphism-inspired UI
- Fully responsive
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
elastic-slide-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser and hover over the button to view the elastic slide popover animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Animations are disabled automatically when **Reduce Motion** is enabled.
