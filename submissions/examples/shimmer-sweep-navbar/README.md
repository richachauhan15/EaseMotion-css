# Shimmer Sweep Navbar

**Issue:** #54283

## Overview

A responsive navigation bar featuring a smooth shimmer sweep animation on hover. Built entirely with HTML and CSS without JavaScript.

## Features

- Pure HTML & CSS
- Shimmer sweep hover animation
- Responsive design
- Keyboard accessible
- Mobile friendly
- `prefers-reduced-motion` support
- Lightweight and reusable

## Folder Structure

```
submissions/examples/shimmer-sweep-navbar/
```

## Files

- style.css
- README.md

## Usage

```html
<link rel="stylesheet" href="style.css">

<nav class="em-navbar">
    ...
</nav>
```
