# Ease Pod Disruption Budget Monitor

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes Pod Disruption Budget (PDB) status.

## Features

- PDB availability overview
- Safe eviction monitoring
- Availability warning detection
- Blocked disruption alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable reliability dashboard for tracking Pod Disruption Budget compliance, supporting safe node maintenance and high availability across Kubernetes clusters.
