# 🧲 Magnet Nav – Items Attract Cursor

> An interactive, playful navigation where items pull toward your cursor like a magnet, creating a responsive, engaging experience.

---

## 📖 Description

The **Magnet Nav** is an interactive navigation component where each item attracts the cursor like a magnet. As you move your mouse near an item, it pulls toward the cursor with a smooth, responsive motion, creating a playful, engaging experience. The magnetic pull is stronger when the cursor is closer, with a subtle scaling effect that makes the interaction feel alive and responsive.

The component features a horizontal navigation bar with five items, each with a unique color icon. On hover, items follow the cursor with a magnetic pull, scale up slightly, and show a tooltip label. The effect uses JavaScript to calculate cursor distance and apply transforms, creating a seamless, interactive experience.

Perfect for creative portfolios, interactive interfaces, playful brand experiences, and any application where you want to add a sense of magic and interactivity to your navigation. The Magnet Nav transforms a simple menu into a captivating, responsive experience.
