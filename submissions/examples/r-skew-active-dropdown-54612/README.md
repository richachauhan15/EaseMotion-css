# CSS Skew-Active Dropdown

A modern **Skew-Active Dropdown** built using pure HTML5 and CSS3 for creative portfolio layouts. The dropdown smoothly transitions from a skewed state into a normal layout on hover or keyboard focus, providing a clean and engaging navigation experience without JavaScript.

---

## ✨ Features

- Pure HTML5 & CSS3
- Skew-to-flat dropdown animation
- Smooth hover and focus interactions
- Responsive navigation layout
- CSS Custom Properties
- Keyboard accessibility using `:focus-within`
- Lightweight and performant
- `prefers-reduced-motion` support
- No JavaScript required
