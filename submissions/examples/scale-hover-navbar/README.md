# Scale-Hover Navbar

Issue: #54287

## Overview

A responsive navigation bar with smooth scale-hover animation built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth scale animation
- Keyboard accessible
- Responsive layout
- Mobile friendly
- Supports `prefers-reduced-motion`
- No JavaScript required

## Folder

```
submissions/examples/scale-hover-navbar/
```

## Files

## Usage

```html
<nav class="em-scale-navbar">
    ...
</nav>
```

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```
