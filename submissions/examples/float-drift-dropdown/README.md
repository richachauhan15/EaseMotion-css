# CSS Float-Drift Dropdown

A responsive dropdown menu built with pure HTML and CSS featuring a smooth floating drift animation.

## Features

- Pure HTML & CSS
- Float-Drift Animation
- Responsive Design
- Keyboard Accessible (`:focus-within`)
- CSS Variables
- `prefers-reduced-motion` Support
- No JavaScript Required

## Folder Structure

```text

```

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

- Supports keyboard navigation using `:focus-within`.
- Supports `prefers-reduced-motion` to reduce animation for users who request it.
