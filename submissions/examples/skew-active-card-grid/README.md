# Skew-Active Card Grid

A responsive card grid featuring a smooth skew hover animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive CSS Grid
- Smooth skew active hover effect
- Hardware accelerated transforms
- Accessible with `prefers-reduced-motion`
- Easy customization using CSS variables

## Folder

```
submissions/examples/skew-active-card-grid/
```

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="sa-grid">
    <div class="sa-card">
        <h3>Title</h3>
        <p>Description</p>
    </div>
</div>
```

## CSS Variables

```css
--bg
--card
--primary
--text
--radius
--duration
```

## Accessibility

- Fully responsive
- Supports `prefers-reduced-motion`
- Keyboard-friendly HTML structure
- No JavaScript required

## Why EaseMotion CSS?

This component follows the EaseMotion CSS philosophy by providing a lightweight, reusable, animation-first card grid using only HTML and CSS.
