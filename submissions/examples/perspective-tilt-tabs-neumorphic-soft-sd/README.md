# Perspective Tilt Tabs

1. A pure CSS tab component with a soft neumorphic appearance and a subtle 3D perspective tilt on the active tab.

2. Include `style.css` and use the `.perspective-tilt-tabs` wrapper with radio inputs, labels, and content divs. Five custom properties are exposed for duration, easing, tilt angle, border radius, and scale.

3. Adds tactile depth to tab interactions without JavaScript while keeping markup minimal and accessible.
