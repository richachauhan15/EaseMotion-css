# ✨ Border Glow Card – Border Animates with Gradient

> A stunning, modern card with an animated gradient border that flows and glows on hover, creating a premium, futuristic visual effect.

---

## 📖 Description

The **Border Glow Card** is a premium UI component that features an animated, multi-color gradient border that flows around the card. On hover, the border becomes more vibrant and speeds up its animation, while the card lifts and the inner content subtly shifts. The effect uses a conic gradient with `mask-composite` to create a clean, glowing border that appears to orbit around the card.

The component features a dark, modern design with a subtle card lift on hover, an animated icon with rotation, and a clean typography layout. The gradient animation cycles through a vibrant palette of blue, cyan, green, pink, and purple, creating a mesmerizing, premium feel.

Perfect for premium product showcases, feature highlights, subscription cards, and any application where you want to create a sense of luxury and sophistication. The Border Glow Card transforms a simple card into a stunning visual experience that captivates users.
