# CSS Ripple Wave Carousel

**Issue:** #54459

## Description

A modern portfolio carousel card layout with a ripple-wave hover animation created using only HTML and CSS.

## Features

- Pure HTML & CSS
- Ripple wave hover effect
- Responsive layout
- No JavaScript
- Accessible with `prefers-reduced-motion`
- Smooth transitions

## Usage

```html
<div class="carousel">

    <div class="card active">
        <h2>Portfolio One</h2>
        <p>Creative UI Design</p>
    </div>

    <div class="card">
        <h2>Portfolio Two</h2>
        <p>Web Development</p>
    </div>

</div>
```

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

## Customization

- Card width
- Background colors
- Ripple size
- Animation duration
- Border radius

## Browser Support

- Chrome
- Firefox
- Edge
