# Team Member Card Component

## Description

A reusable Team Member Card component built using HTML and CSS. It displays a profile image, member name, role, short description, and social media links in a clean and responsive card layout.

## Features

- Responsive card layout
- Circular profile image
- Role and description section
- Social media links
- Smooth hover animation
- Pure HTML & CSS

## Usage

```html
<div class="team-card">
    <img src="profile.jpg" alt="Team Member">
    <h2>John Doe</h2>
    <p class="role">Frontend Developer</p>
    <p class="description">
        Passionate about building responsive web applications.
    </p>

    <div class="social-links">
        <a href="#">GitHub</a>
        <a href="#">LinkedIn</a>
        <a href="#">Twitter</a>
    </div>
</div>
```
