# ⌨️ Typewriter Button – Self-Typing Text

> A nostalgic, engaging button where text types itself out character by character on hover, creating a classic typewriter effect that captivates users.

---

## 📖 Description

The **Typewriter Button** is a charming UI component that simulates a classic typewriter effect on hover. The text appears character by character with realistic typing delays (50-150ms), complete with a blinking cursor that follows the typing progress. The effect creates a nostalgic, engaging interaction that feels both familiar and delightful.

The component features a retro-inspired design with a dark, mechanical aesthetic reminiscent of vintage typewriters. The cursor blinks at the end of the text, and the keyboard icon taps along with the typing rhythm. On hover, the text types itself out, and on mouse leave, it resets to its initial state, ready to type again.

Perfect for nostalgic interfaces, retro-themed applications, educational content, and any interaction where you want to create a sense of anticipation and engagement. The Typewriter Button transforms a simple hover into a memorable, story-driven experience.
