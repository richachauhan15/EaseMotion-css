# Checkbox Print Optimization

This component provides a custom-styled checkbox designed to save ink and maintain high legibility when printed.

## Features
- **Ink Saving:** Automatically strips heavy background gradients, colorful fills, and box-shadows when a print command is triggered.
- **High Contrast:** Converts borders and checkmarks to clear black-and-white print styles using `@media print`.

## Usage
Open `demo.html` in your browser and use your browser's Print Preview feature (`Ctrl+P` or `Cmd+P`) to see the print optimization in action.
