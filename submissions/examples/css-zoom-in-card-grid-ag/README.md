# CSS Zoom-In Card Grid

A lightweight, pure CSS showcase example demonstrating a smooth zoom-in image effect on hover, perfect for creative portfolio layouts and image galleries.

## Features
- **Pure CSS:** No JavaScript required.
- **Responsive Grid:** Uses CSS Grid with `auto-fit` for fluid responsiveness.
- **Smooth Animation:** Utilizes `cubic-bezier` transitions for a premium feel.
- **Accessibility Ready:** Includes `prefers-reduced-motion` support for users sensitive to animations.

## Usage
1. Link the `style.css` in your HTML `<head>`.
2. Wrap your cards in a container with the `.ease-zoom-grid` class.
3. Use the `.ease-zoom-card` class for individual items and `.image-wrapper` for the image container to ensure the zoom effect stays within the card boundaries.
