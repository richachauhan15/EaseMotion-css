# CSS Skew-Active Pricing Table

A modern **Skew-Active Pricing Table** built using only HTML and CSS for creative portfolio layouts.

---

## Features

- Pure HTML5 & CSS3
- Skew-active hover animation
- Smooth transitions
- Responsive pricing grid
- Featured pricing card
- CSS Custom Properties
- Accessible HTML
- `prefers-reduced-motion` support
- No JavaScript required

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` in any modern browser to view the component.
