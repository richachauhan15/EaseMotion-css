# 📝 Peel Card – Corner Peels Like Sticky Note

> A charming, interactive card where the corner peels back like a sticky note, revealing hidden content underneath with a satisfying 3D effect.

---

## 📖 Description

The **Peel Card** is a delightful UI component that simulates the peeling of a sticky note corner. On hover, the bottom-right corner lifts and folds back, revealing hidden content beneath the card. The effect uses CSS 3D transforms and clip-paths to create a realistic peeling motion that mimics real sticky notes.

The component features a warm, sticky-note aesthetic with a yellow front and a green back. The corner fold is accented with a subtle shadow, and the entire card shifts slightly to create depth. The hidden content is revealed smoothly with a fade-in effect, and a close button allows users to unpeel the card.

Perfect for interactive storytelling, surprise reveals, sticky note widgets, and any application where you want to create a sense of discovery and delight. The Peel Card transforms a simple card into an engaging, tactile experience that users will love.
