# CSS Skew-Active Carousel

Issue: #54447

## Overview

A responsive portfolio carousel where the active card removes its skew and scales smoothly, creating a modern focus effect using only HTML and CSS.

## Features

- Pure HTML & CSS
- Skew-to-normal hover animation
- Smooth transitions
- Responsive layout
- Accessible
- Supports prefers-reduced-motion
- No JavaScript required

## Usage

```html
<div class="carousel">
    <div class="card active">...</div>
    <div class="card">...</div>
    <div class="card">...</div>
</div>
```
