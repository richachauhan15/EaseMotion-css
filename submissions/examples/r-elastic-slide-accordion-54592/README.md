# CSS Elastic-Slide Accordion

A modern **Elastic-Slide Accordion** built using pure HTML5 and CSS3 for creative portfolio layouts. It uses the native `<details>` and `<summary>` elements to provide an accessible accordion with smooth elastic slide animations, responsive design, and zero JavaScript.

---

## ✨ Features

- Pure HTML5 & CSS3
- Native `<details>` / `<summary>` accordion
- Elastic slide opening animation
- Smooth expand and collapse transitions
- Responsive layout
- CSS Custom Properties
- Accessible by default
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in your browser.
