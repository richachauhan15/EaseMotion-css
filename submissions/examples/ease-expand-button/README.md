# 🔍 Expand Button – Search Bar Expansion

> A sleek button that smoothly expands into a full search bar on click, with a satisfying elastic animation and intuitive close functionality.

---

## 📖 Description

The **Expand Button** is an elegant UI component that transforms a compact button into a fully functional search bar on click. The button expands outward with a smooth, elastic animation, revealing a search input field with a close button. This space-saving design is perfect for navigation bars, headers, and any interface where you want to provide search functionality without taking up permanent screen real estate.

The component features a clean, modern design with a subtle gradient button that morphs into a white search bar. The expansion uses a `cubic-bezier` easing for a spring-like feel, making the interaction feel natural and satisfying. The search bar includes a close button, keyboard support (Escape to close), and click-outside-to-close functionality, making it both intuitive and accessible.
