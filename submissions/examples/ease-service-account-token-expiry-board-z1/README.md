# Ease Service Account Token Expiry Board

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes Service Account Token validity and expiration.

## Features

- Token validity overview
- Token rotation monitoring
- Expiration warning alerts
- Expired credential detection
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable Kubernetes security component for tracking service account credentials, simplifying token lifecycle management, and improving cluster security posture.
