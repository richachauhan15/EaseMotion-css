# CSS-only Interactive Radial Action Menu

A modern **CSS-only Interactive Radial Action Menu** that expands multiple circular action buttons from a central floating button using smooth transitions and transforms.

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Smooth radial expansion animation
- Staggered transition delays
- Hover-based interaction
- Responsive layout
- Easy to customize

## 📁 Folder Structure

```
submissions/examples/radial-action-menu/
├── demo.html
├── style.css
└── README.md
```

## 🚀 How to Run

1. Open `demo.html` in any modern browser.
2. Hover over the center button.
3. The action buttons expand outward with smooth animations.

## 🎨 Customization

You can easily customize:

- Colors
- Icons
- Number of action buttons
- Animation duration
- Transition delay
- Expansion distance
- Button size

## 📦 Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Transforms
- CSS Transitions

Hover over the center button to reveal six animated action buttons arranged in a circular pattern.
