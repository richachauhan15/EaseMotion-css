# Ease Network Policy Enforcement Status

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes NetworkPolicy enforcement and namespace security posture.

## Features

- NetworkPolicy status overview
- Policy rollout monitoring
- Namespace security visibility
- Enforcement failure alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable cloud security component for visualizing NetworkPolicy enforcement, identifying unsecured workloads, and improving zero-trust networking across Kubernetes clusters.
