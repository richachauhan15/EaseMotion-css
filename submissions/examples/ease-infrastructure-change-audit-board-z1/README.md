# Ease Infrastructure Change Audit Board

## Overview

A responsive HTML and CSS component for monitoring infrastructure changes, approval workflows, compliance checks, and unauthorized modifications.

## Features

- Infrastructure change tracking
- Approval workflow visibility
- Compliance monitoring
- Unauthorized change alerts
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable dashboard component for infrastructure governance, compliance auditing, change management, and cloud operations.
