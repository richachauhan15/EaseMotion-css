# 🖼️ Hover Zoom Card – Image Zooms, Text Overlays

> A stunning card where the image zooms in smoothly on hover while a text overlay fades in, creating an elegant, interactive showcase for visual content.

---

## 📖 Description

The **Hover Zoom Card** is an elegant UI component that showcases images with a smooth zoom effect and text overlay on hover. The image scales up with a spring-like motion, while a gradient overlay and text content fade in from the bottom, creating a polished, interactive experience perfect for portfolios, galleries, and product showcases.

The component features a clean card design with an image, title, description, and tags. On hover, the image zooms in (scale 1.15), the overlay appears with a fade effect, and the text content slides up with a bounce. The footer includes category and likes information, with a subtle heart beat animation on hover.
