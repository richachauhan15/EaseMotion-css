# Ripple Wave Accordion

A responsive accordion component featuring a ripple-wave opening animation built using only HTML and CSS.

## Features

- Pure HTML & CSS
- Native `<details>` and `<summary>`
- Ripple-wave opening animation
- Responsive design
- Accessible
- Supports `prefers-reduced-motion`
- No JavaScript required

## Usage

```html
<link rel="stylesheet" href="style.css">

<details class="em-accordion">
    <summary>Accordion Title</summary>
    <p>Accordion content.</p>
</details>
```
