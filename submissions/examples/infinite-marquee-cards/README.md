# Infinite Marquee Cards

A smooth **CSS-only Infinite Marquee Cards** component that continuously scrolls cards horizontally.

## Features

- ✨ Pure HTML & CSS
- ♾️ Infinite scrolling animation
- ⏸️ Pause on hover
- 🎨 Smooth hover lift effect
- 📱 Responsive layout
- 🚫 No JavaScript required

---

## Preview

The cards scroll continuously from right to left. Hovering over the marquee pauses the animation, allowing users to interact with individual cards.

## Usage

1. Open `demo.html` in your browser.
2. Customize the card content.
3. Adjust the animation duration inside `style.css` if needed.

```css
animation: marquee 18s linear infinite;
```

Increase the duration for slower scrolling or decrease it for faster scrolling.
