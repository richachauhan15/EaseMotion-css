# Skew-Active Accordion

**Issue:** #54327

## Overview

A modern accordion featuring a subtle skew animation whenever an item becomes active. Built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Smooth skew animation
- Uses semantic `<details>` and `<summary>`
- Keyboard accessible
- No JavaScript required
- Supports `prefers-reduced-motion`

## Usage

```html
<link rel="stylesheet" href="style.css">

<details class="accordion">
    <summary>Accordion Title</summary>
    <div class="content">
        Your content here.
    </div>
</details>
```
