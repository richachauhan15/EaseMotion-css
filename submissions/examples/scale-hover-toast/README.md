# Scale Hover Toast

A responsive toast notification component featuring a subtle scale hover animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth scale hover effect
- Responsive layout
- Accessible
- Supports `prefers-reduced-motion`
- No JavaScript required

## Folder

```
submissions/examples/scale-hover-toast/
```

## Files

```
demo.html
style.css
README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="em-toast success">
    <span class="icon">✔</span>

    <div>
        <h3>Success</h3>
        <p>Notification message</p>
    </div>
</div>
```

### Available Variants

- Success
- Warning
- Error
