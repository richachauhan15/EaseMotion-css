# CSS Slide-Up Hero Section

A responsive Hero Section built using pure HTML and CSS featuring a smooth slide-up entrance animation.

## Features

- Pure HTML & CSS
- Slide-Up Entrance Animation
- Responsive Layout
- CSS Variables
- Accessible
- prefers-reduced-motion Support
- No JavaScript Required

## Folder Structure

```
slide-up-hero-section/
├── demo.html
├── style.css
└── README.md
```

## CSS Variables

```css
--primary
--secondary
--background
--text
--white
--radius
--transition
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports the `prefers-reduced-motion` media query to reduce or disable animations for users who prefer reduced motion.
