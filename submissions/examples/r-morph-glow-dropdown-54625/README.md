# CSS Morph-Glow Dropdown

A modern **Morph-Glow Dropdown** built using pure HTML5 and CSS3 for creative portfolio layouts. The component features a smooth morphing scale animation, soft glow effects, keyboard accessibility, and responsive behavior without requiring JavaScript.

---

## ✨ Features

- Pure HTML5 & CSS3
- Morph-glow dropdown animation
- Frosted glass dropdown panel
- Smooth hover and focus interactions
- Keyboard accessible using `:focus-within`
- CSS Custom Properties
- Fully responsive layout
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

3. Open `demo.html` in any modern browser.
