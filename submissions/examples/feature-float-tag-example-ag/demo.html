Quick Integration Guide

Apply the class float-tag-example-ag or float-tag-ag to any element to enable the hover float effect: 

<!-- Minimal Example -->
<div class="float-tag-example-ag" tabindex="0">Hover me</div>

<!-- Fully Accessible Button Tag -->
<button class="float-tag-ag float-tag-primary-ag" type="button">
  #UI Design
</button>
