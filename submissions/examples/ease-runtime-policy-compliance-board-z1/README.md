# Ease Runtime Policy Compliance Board

## Overview

A responsive HTML and CSS dashboard component for visualizing runtime policy compliance across cloud-native workloads.

## Features

- Runtime compliance monitoring
- Policy validation status
- Security violation alerts
- Exception tracking
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component helps DevOps and security teams quickly identify runtime policy violations, monitor compliance status, and improve Kubernetes security posture using a lightweight, reusable UI.
