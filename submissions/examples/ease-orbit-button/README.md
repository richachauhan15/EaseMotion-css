# 🪐 Orbit Button – Orbiting Dots Effect

> A cosmic, mesmerizing button where small dots orbit around it like planets around a sun, creating a dynamic, space-inspired interaction.

---

## 📖 Description

The **Orbit Button** is a stunning UI component that simulates planets orbiting around a central button. Eight colorful dots orbit the button in smooth, circular paths, each with its own speed, color, and timing. The effect creates a cosmic, space-inspired interaction that's both mesmerizing and engaging.

The component features a dark, space-themed design with a central button glowing like a star. The orbiting dots have individual colors and speeds, creating a dynamic, organic feel. On hover, the dots orbit continuously, and on click, they burst outward in a satisfying pulse. The button itself responds with a subtle scale and glow effect.

Perfect for space-themed applications, gaming interfaces, creative portfolios, and any interaction that needs to feel cosmic and dynamic. The Orbit Button transforms a simple button into a miniature solar system that captivates users.
