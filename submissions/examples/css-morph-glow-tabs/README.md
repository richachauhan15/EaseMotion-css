# CSS Morph Glow Tabs

A responsive Morph Glow Tabs component built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Morphing glow animation
- Responsive layout
- No JavaScript
- Accessible (`prefers-reduced-motion`)
- Easy to customize

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="tabs">
    <!-- Radio buttons -->
    <!-- Labels -->
    <!-- Content -->
</div>
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports users who prefer reduced motion through the `prefers-reduced-motion` media query.
