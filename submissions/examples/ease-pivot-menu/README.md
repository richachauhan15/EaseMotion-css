# 🔄 Pivot Menu – Rotates to Reveal Sub-items

> A sleek, mechanical menu where the toggle rotates to reveal sub-items beneath it, creating a satisfying pivot effect with staggered animation.

---

## 📖 Description

The **Pivot Menu** is a sleek UI component where the toggle button rotates to reveal sub-items beneath it with a smooth, mechanical pivot effect. The menu opens with a cascading, staggered animation, where each sub-item appears sequentially with a spring-like bounce. The toggle icon rotates 180 degrees when the menu is open, providing clear visual feedback.

The component features a clean, modern design with a gradient toggle button and minimal sub-items with icons. The pivot indicator line adds a subtle mechanical touch, connecting the toggle to the sub-items. Each sub-item has a unique icon color and responds with hover and click feedback.

Perfect for admin dashboards, settings panels, navigation menus, and any application where you want a compact, mechanical menu interaction. The Pivot Menu transforms a simple dropdown into a satisfying, polished experience.
