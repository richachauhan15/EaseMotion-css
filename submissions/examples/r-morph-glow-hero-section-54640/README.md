# CSS Morph-Glow Hero Section

A modern **Morph-Glow Hero Section** built using pure HTML5 and CSS3 for creative portfolio layouts. This example showcases animated glowing background elements with smooth morphing effects, responsive design, and accessible motion preferences.

---

## ✨ Features

- Pure HTML5 & CSS3
- Morphing glow background animation
- Smooth fade-up entrance animation
- Gradient call-to-action button
- Responsive hero layout
- CSS Custom Properties
- Modern dark UI
- `prefers-reduced-motion` support
- No JavaScript required

## 🚀 Usage

1. Copy the component into your project.
2. Link the stylesheet:

```html

```

3. Open `demo.html` in any modern browser.
