# 🍿 Popcorn Button – Popping Dots Effect

> A playful, explosive button where small dots pop outward like popcorn on hover or click, creating a delightful, energetic interaction.

---

## 📖 Description

The **Popcorn Button** is a fun, energetic UI component that simulates popcorn popping on hover or click. Twelve colorful dots burst outward from the button in various directions, creating an explosive, playful effect that captures attention and brings joy to the interaction. The dots follow individual trajectories with staggered timing, making the effect feel organic and lively.

The component features a warm, inviting color palette inspired by popcorn – golden yellow and orange gradients. The button content includes a popcorn icon and "Pop!" text, with the icon shaking playfully on hover. The dots pop with a satisfying spring-like motion, using `cubic-bezier` easing for a natural, bouncy feel.

Perfect for entertainment apps, kids' interfaces, playful marketing materials, and any application where you want to add a touch of fun and energy to the user experience. The Popcorn Button transforms a simple interaction into a moment of delight that users will remember.
