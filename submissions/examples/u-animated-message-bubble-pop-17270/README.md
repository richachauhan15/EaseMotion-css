# Animated Message Bubble Pop

## Overview

Animated Message Bubble Pop is a lightweight CSS animation that creates a subtle popping effect for chat messages.

## Features

- CSS-only animation
- Smooth pop effect
- Messaging UI example
- Responsive design
- Accessibility friendly
- Easy to customize

## Usage

```html
<div class="message pop">
  New message received!
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
