# CSS Blur Entrance Hero Section

A responsive Hero Section built with pure HTML and CSS featuring a smooth blur entrance animation for modern landing pages.

## Features

- Pure HTML & CSS
- Blur Entrance Animation
- Responsive Layout
- CSS Variables
- Accessible
- prefers-reduced-motion Support
- No JavaScript Required

## Folder Structure

```
blur-entrance-hero-section/
├── demo.html
├── style.css
└── README.md
```

## CSS Variables

```css
--primary
--secondary
--background
--text
--white

--transition
```

## Usage

Open `demo.html` in any modern web browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports the `prefers-reduced-motion` media query to disable animations for users who prefer reduced motion.
