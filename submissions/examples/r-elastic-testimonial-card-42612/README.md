# Elastic Testimonial Card

A modern **Elastic Testimonial Card** built with pure CSS for the **EaseMotion CSS** library. The component features a smooth elastic entrance and hover animation inspired by **Neumorphism** design, making testimonials feel interactive while remaining lightweight and accessible.

---

## ✨ Features

- Pure CSS implementation
- Elastic entrance animation
- Elastic hover interaction
- Neumorphism-inspired design
- Responsive layout
- Keyboard accessible
- No JavaScript required
- Easy customization using CSS variables
