\# 3D-Flip Pricing Table Component (`ease-pricing-3d-ak`)



A modern, responsive CSS 3D-flip pricing table designed specifically for creative portfolios, built cleanly for \*\*EaseMotion CSS\*\*.



\## Features

\- \*\*Pure CSS 3D Perspective\*\*: Utilizes `perspective`, `transform-style: preserve-3d`, and `backface-visibility: hidden` for butter-smooth rotation.

\- \*\*Glassmorphism Design\*\*: Styled with modern backdrop blurs, subtle borders, and flexible layout cards.

\- \*\*Accessibility Ready\*\*: Fully respects user motion preferences via `@media (prefers-reduced-motion: reduce)`.

\- \*\*Zero JS Dependencies\*\*: Built entirely using modern native CSS and lightweight EaseMotion utility classes.



\## Quick Usage

1\. Place the folder inside your repository:

&#x20;  `submissions/examples/pricing-table-3d-ak/`

2\. Include the stylesheet and demo HTML in your project to view the interactive flip effect.
