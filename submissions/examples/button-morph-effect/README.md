# Button Morph Effect

## Description

Button Morph Effect is a reusable hover utility that smoothly changes a button's shape, size, and shadow when hovered. It creates a modern call-to-action effect using only HTML and CSS.

## Features

- Smooth morphing animation
- Rounded shape transition
- Scale effect on hover
- Enhanced shadow
- Pure HTML & CSS
- Reusable utility

## Usage

```html
<button class="button-morph-effect">
    Hover Me
</button>
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
