# 🎯 Radial Menu – Items Orbit Out from Center

> A sleek, interactive menu where items orbit out from the center in a circular pattern with staggered, spring-like animations.

---

## 📖 Description

The **Radial Menu** is a dynamic UI component where menu items orbit outward from a central toggle button in a circular pattern. Each item appears with a staggered delay and a spring-like animation, creating an organic, engaging experience. The center button morphs from a plus (+) icon to a close (×) icon when the menu is open.

The component features six menu items arranged in a circle, each with a unique color gradient and tooltip label. Items have hover and click feedback, and the menu closes automatically when an item is selected or when clicking outside. The staggered animation (50ms delays) creates a natural, sequential reveal that feels polished and intentional.

Perfect for mobile navigation, action menus, creative interfaces, and any application where you want a visually striking, space-efficient menu. The Radial Menu transforms a simple navigation into a captivating, interactive experience.
