# 🎯 Perspective Nav – 3D Depth on Hover

> A stunning navigation menu where each item pops with 3D depth and perspective transforms on hover, creating an immersive, interactive experience.

---

## 📖 Description

The **Perspective Nav** is a visually stunning navigation component where each menu item gains 3D depth on hover using CSS perspective transforms. Items rotate and translate in 3D space, creating a sense of depth and immersion that makes the navigation feel alive and interactive.

The component features individual 3D transforms for each navigation item, with varied rotation directions for visual variety. Items include icons and labels, with glow effects, shadow drops, and tooltip labels on hover. The navigation uses `perspective: 800px` to create realistic 3D depth, and `transform-style: preserve-3d` to enable 3D child elements.

Perfect for creative portfolios, modern interfaces, and any application where you want to add a sense of depth and interactivity to your navigation. The Perspective Nav transforms a simple menu into an immersive, engaging experience.
