# Social Icons Overflow Fix

## Description

This fix prevents social media links from overflowing outside the card on small screens.

## Changes Made

- Added `flex-wrap: wrap` to the social links container.
- Improved spacing using `gap`.
- Added responsive styles for screens below 320px.
- Ensures icons remain inside the card without breaking the layout.
