# CSS 3D Flip Drawer

A responsive 3D Flip Drawer built using only HTML and CSS.

## Features

- Pure HTML & CSS
- 3D flip animation
- Glassmorphism design
- Responsive layout
- No JavaScript
- Accessible with `prefers-reduced-motion`

## Folder Structure

```
3d-flip-drawer/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in a browser.

Hover over the drawer to see the 3D flip animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Animations are disabled automatically for users who enable **Reduce Motion**.
