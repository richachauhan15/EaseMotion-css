# CSS Zoom-In Drawer

A responsive Zoom-In Drawer built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Smooth zoom-in entrance animation
- Glassmorphism-inspired design
- Responsive layout
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
zoom-in-drawer/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in your browser.

The drawer smoothly zooms into view when the page loads.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Users who enable **Reduce Motion** will automatically receive a non-animated version.
