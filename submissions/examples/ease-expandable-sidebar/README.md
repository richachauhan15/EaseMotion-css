# 📂 Expandable Sidebar – Collapses with Icon Only

> A sleek, space-saving sidebar that collapses to show only icons, providing a modern, efficient navigation experience for dashboards and admin interfaces.

---

## 📖 Description

The **Expandable Sidebar** is a sleek, space-saving navigation component that collapses to show only icons when not in use. The sidebar smoothly transitions from a full-width navigation (with text labels) to a compact icon-only version, maximizing screen real estate for content. A toggle button animates with a spring-like motion, providing a satisfying, polished interaction.

The component features a dark, modern design with gradient colors, active state highlighting, hover effects, and tooltips that appear when the sidebar is collapsed. It includes a header with brand logo, navigation links with icons and text, a badge indicator, and a footer with a logout option. The sidebar is fully responsive, adapting to different screen sizes with a mobile-friendly slide-out version.

Perfect for admin dashboards, web applications, CMS interfaces, and any application where you need a space-efficient, professional navigation experience. The Expandable Sidebar transforms a simple menu into a polished, functional component that enhances user experience.
