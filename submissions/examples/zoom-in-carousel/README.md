# CSS Zoom-In Carousel

**Issue:** #54451

## Description

A responsive portfolio carousel featuring a smooth zoom-in hover effect using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Zoom-in hover animation
- Responsive layout
- No JavaScript required
- Smooth transitions
- Accessibility support with `prefers-reduced-motion`

## Usage

```html
<div class="carousel">

    <div class="card active">
        <h2>Project One</h2>
        <p>Creative Portfolio Design</p>
    </div>

    <div class="card">
        <h2>Project Two</h2>
        <p>Responsive Website</p>
    </div>

</div>
```

Include:

```html
<link rel="stylesheet" href="style.css">
```

## Customization

- Card width
- Background color
- Zoom scale
- Shadow intensity
- Border radius
- Transition duration

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
