# Float Drift Stepper

A lightweight pure CSS stepper component with a smooth floating drift animation.

## Features

- Pure HTML and CSS
- No JavaScript required
- Smooth floating animation
- Responsive layout
- Accessible reduced-motion support

## Usage

Add the stepper structure:

```html
<div class="stepper">

  <div class="step">
    <span>1</span>
    <h3>Step Title</h3>
  </div>

</div>
