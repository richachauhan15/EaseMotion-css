# 📷 Shutter Button – Camera Aperture Effect

> A dramatic button that opens like a camera aperture on hover, with multiple blades retracting to reveal the center in a stunning iris effect.

---

## 📖 Description

The **Shutter Button** is a cinematic UI component that simulates a camera lens aperture opening on hover. Eight triangular blades arranged in a circular pattern retract outward, creating a stunning iris effect that reveals the button's content at the center. The dark, sophisticated design mimics professional camera lenses, complete with subtle metallic gradients and shadows.

This component is perfect for photography apps, creative portfolios, and any interface where you want to capture attention with a dramatic, mechanical interaction. The aperture opening creates a sense of precision and professionalism, making it ideal for camera-related actions, capture buttons, or any interaction that requires a moment of focus.
