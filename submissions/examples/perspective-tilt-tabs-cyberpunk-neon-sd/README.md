# Perspective Tilt Tabs — Cyberpunk Neon

## What does this do?

CSS-only tabbed interface with a subtle 3D perspective tilt effect on the active tab. Cyberpunk neon glow styling using custom properties.

## How is it used?

Include `style.css` and use the `.perspective-tilt-tabs` markup pattern. Customize via CSS custom properties:

- `--tabs-duration` — transition speed
- `--tabs-easing` — timing function
- `--tabs-tilt-angle` — 3D rotation amount
- `--tabs-neon-color` — glow color
- `--tabs-scale` — active tab scale

## Why is it useful?

Provides a neon-styled tab component with zero JavaScript, keyboard accessibility via radio inputs, and configurable 3D tilt feedback — ready for cyberpunk-themed projects.
