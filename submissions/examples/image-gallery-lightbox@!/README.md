# Image Gallery with Lightbox

This is a self-contained example demonstrating how to create a modern, responsive image gallery with a pure CSS lightbox modal using the checkbox hack technique, seamlessly integrated with **EaseMotion CSS** utility classes.

## Features
- 🖼️ **Responsive Grid Layout:** Auto-filling grid that adapts to screen size.
- 🎨 **Hover Effects:** Image zoom, overlay fade-in, and animated zoom icon.
- 🔍 **Pure CSS Lightbox:** No JavaScript required for opening/closing modals.
- ✨ **Smooth Animations:** Scale and fade transitions for lightbox.
- ⌨️ **Keyboard Accessible:** ESC key to close lightbox.
- ♿ **Fully Accessible:** Proper ARIA labels, focus states, and reduced motion support.
- 📱 **Responsive Design:** Adapts from 3 columns on desktop to 2 on mobile.
- 🎯 **Click Outside to Close:** Click on backdrop to dismiss lightbox.
- 🚫 **Zero JavaScript for Core Functionality:** Only minimal JS for ESC key and backdrop click.

## How to Use
1. Ensure the EaseMotion CSS CDN is linked in your `<head>`.
2. Link the `style.css` file.
3. Copy the gallery HTML structure.
4. Replace image URLs with your own.
5. Include the minimal JavaScript for ESC key support (optional).
