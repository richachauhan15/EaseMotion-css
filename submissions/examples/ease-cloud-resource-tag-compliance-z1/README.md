# Ease Cloud Resource Tag Compliance

## Overview

A responsive HTML and CSS dashboard component for monitoring cloud resource tagging compliance and governance.

## Features

- Resource tag compliance
- Missing owner detection
- Policy violation alerts
- Deployment readiness
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open **demo.html** in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component helps DevOps and FinOps teams maintain tagging standards, improve cost allocation, enforce governance policies, and simplify cloud resource management.
