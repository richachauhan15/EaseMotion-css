# ✂️ Slice Button – Split & Reveal

> A dramatic button that splits into two halves on hover, revealing hidden text beneath with a smooth, cinematic animation.

---

## 📖 Description

The **Slice Button** is an engaging UI component that creates a dramatic "slicing" effect on hover. The button cleanly separates into two halves – the top sliding upward and the bottom sliding downward – revealing hidden text that was concealed behind the button. This creates a satisfying "cut" or "slice" metaphor that's perfect for actions involving division, separation, or revelation.

The effect is achieved using CSS transforms with a spring-like cubic-bezier easing, giving the split a natural, bouncy feel. A subtle glowing line appears at the cut point, reinforcing the slicing metaphor, while the hidden text fades in with a slight scale effect for added polish.
