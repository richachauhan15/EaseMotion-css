# Fade Scale Hover Utility

## Description

Fade Scale Hover Utility provides a smooth hover animation where an element gently scales up while slightly fading. It is useful for cards, buttons, feature sections, and interactive UI components.

## Features

- Smooth scale animation
- Subtle fade effect
- Lightweight and reusable
- Pure HTML & CSS
- Easy to customize

## Usage

```html
<div class="fade-scale-effect">
    <h2>Hover Me</h2>
    <p>Fade & Scale Hover Utility</p>
</div>
```
