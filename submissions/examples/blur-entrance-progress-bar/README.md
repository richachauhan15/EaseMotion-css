# CSS Blur Entrance Progress Bar

**Issue:** #54439

## Overview

A modern progress bar with a smooth blur entrance animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Blur entrance animation
- Responsive layout
- Accessible progress bar
- No JavaScript
- Supports prefers-reduced-motion

## Folder

```
submissions/examples/blur-entrance-progress-bar/
```

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="progress">
    <div class="progress-fill">
        <span>80%</span>
    </div>
</div>
```

## Customization

- Progress width
- Animation duration
- Blur amount
- Colors
- Height
- Border radius

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

- Uses `role="progressbar"`
- Responsive layout
- Supports `prefers-reduced-motion`
