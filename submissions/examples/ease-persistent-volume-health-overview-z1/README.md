# Ease Persistent Volume Health Overview

## Overview

A responsive HTML and CSS dashboard component for monitoring Kubernetes Persistent Volume health and storage reliability.

## Features

- Persistent Volume health monitoring
- Volume expansion tracking
- Capacity warning alerts
- Attachment failure visibility
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable storage monitoring component that helps platform teams track volume health, identify storage risks, and improve workload reliability.
