How It Works

Each modal uses a hidden checkbox toggle. When checked, the overlay appears and the modal floats in with a drift animation using cubic-bezier elastic easing. Click cancel, confirm, or the X to close.

Accessibility

Modals include role="dialog", aria-modal="true", and aria-label for screen readers. The prefers-reduced-motion media query disables all animations for users who prefer reduced motion.
