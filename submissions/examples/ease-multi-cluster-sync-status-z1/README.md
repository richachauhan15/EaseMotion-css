# Ease Multi-Cluster Sync Status

## Overview

A responsive HTML and CSS dashboard component for monitoring synchronization status across multiple Kubernetes clusters.

## Features

- Cluster synchronization overview
- GitOps deployment monitoring
- Configuration drift detection
- Synchronization failure alerts
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable GitOps monitoring component for visualizing synchronization health, identifying configuration drift, and improving operational consistency across multiple Kubernetes clusters.
