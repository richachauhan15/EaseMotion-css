# 🫧 Bubble Menu – Items Float Like Bubbles

> A playful, whimsical menu where items float like bubbles with buoyant, floating animations, creating a light and delightful interaction.

---

## 📖 Description

The **Bubble Menu** is a playful UI component where menu items float like bubbles with a buoyant, floating animation. Each bubble gently bobs up and down with its own unique rhythm and delay, creating a whimsical, organic feel. The menu opens from a central toggle button, and each bubble radiates outward with a staggered, spring-like reveal.

The component features six bubble items arranged in a circular pattern, each with its own translucent color and floating animation. Bubbles have a glass-morphism effect with `backdrop-filter: blur(10px)`, creating a light, airy aesthetic. Tooltips appear on hover, and clicking a bubble triggers a satisfying "pop" effect before the menu closes.

Perfect for playful interfaces, children's apps, creative portfolios, and any application where you want to add a touch of whimsy and delight. The Bubble Menu transforms a simple navigation into a magical, engaging experience that brings a smile to users' faces.
