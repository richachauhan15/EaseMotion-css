Float Accordion

A premium, accessible CSS accordion with interactive float hover transitions.

What is EaseMotion CSS?
EaseMotion CSS is a modern, high-performance animation framework curated to build premium, interactive transitions with ease. It features highly optimized CSS classes and SCSS mixins.
How does the float effect work?
The float effect applies a smooth lift and subtle box-shadow transformation on hover using Hardware-Accelerated transitions, paired with a slide-up entrance animation for the content when expanded.
Is this accessible?
Yes! It leverages semantic HTML <details> and <summary> tags for native keyboard navigation and screen reader support, and respects user prefers-reduced-motion media queries.
