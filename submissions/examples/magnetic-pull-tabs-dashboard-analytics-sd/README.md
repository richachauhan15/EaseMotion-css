# Magnetic Pull Tabs

1. **What does this do?**  
   Pure CSS tab component where the active tab subtly moves toward the content panel, creating a magnetic pull effect.

2. **How is it used?**  
   Include `style.css` and use the `.magnetic-pull-tabs` class with radio inputs, labels, and content panels as direct children.

3. **Why is it useful?**  
   Provides a lightweight, accessible, keyboard-friendly tab interface with a smooth visual cue for the active state — no JavaScript required.
