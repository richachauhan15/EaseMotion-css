# CSS Bounce Pulse Tabs

A responsive **Bounce Pulse Tabs** component built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Bounce pulse tab animation
- Responsive design
- No JavaScript required
- Accessible with `prefers-reduced-motion`
- Lightweight and reusable

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="tabs">
    <!-- Radio inputs -->
    <!-- Tab labels -->
    <!-- Tab content -->
</div>
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports `prefers-reduced-motion` for users who prefer reduced animations.
