# Ease Certificate Renewal Pipeline Board

## Overview

A responsive HTML and CSS component for monitoring SSL/TLS certificate renewal pipelines, expiration timelines, and renewal failures.

## Features

- Certificate health overview
- Automated renewal monitoring
- Expiration alerts
- Failed renewal detection
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable infrastructure monitoring component for certificate lifecycle management, reducing service outages caused by expired SSL/TLS certificates and improving cloud security operations.
