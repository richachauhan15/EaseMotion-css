# CSS Rotate Fade Carousel

**Issue:** #54455

## Description

A responsive portfolio carousel featuring a rotate and fade hover animation built using only HTML and CSS.

## Features

- Pure HTML & CSS
- Rotate + fade hover animation
- Responsive layout
- No JavaScript
- Smooth transitions
- Accessible with `prefers-reduced-motion`

## Usage

```html
<div class="carousel">

    <div class="card active">
        <h2>Project One</h2>
        <p>Creative Portfolio</p>
    </div>

    <div class="card">
        <h2>Project Two</h2>
        <p>UI Design</p>
    </div>

</div>
```

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

## Customization

- Card width
- Background colors
- Rotation angle
- Animation duration
- Border radius
- Shadow intensity

## Browser Support

- Chrome
- Firefox
- Edge
