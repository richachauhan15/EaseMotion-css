# Tilt Card Hover Utility

A reusable CSS utility that applies a subtle 3D tilt effect to cards on hover.

## Features

- Smooth 3D tilt animation
- Lightweight and reusable
- Pure CSS
- Modern interactive card effect
- Easy to integrate

## Usage

```html
<div class="tilt-card">
  <h2>Tilt Card</h2>
  <p>Hover over this card.</p>
</div>
```

Include the `style.css` file in your project and apply the `tilt-card` class to any card element.
