# 💧 Liquid Button – Fluid Background Flow

> A mesmerizing button where the background flows like liquid on hover, creating a fluid, wave-like motion with multiple colour layers.

---

## 📖 Description

The **Liquid Button** is a captivating UI component that simulates fluid motion on hover. Three semi-transparent wave layers flow seamlessly behind the button content, each with its own colour gradient, speed, and trajectory. The waves morph and shift organically, creating the illusion of liquid flowing behind the button surface.

This component is perfect for:
- **Creative and artistic interfaces** that need a fluid, organic feel
- **Water, ocean, or nature-themed applications**
- **Brand moments** that convey flow, movement, or transformation
- **Call-to-action buttons** that need to stand out
- **Design portfolios** and creative agencies
- **Interactive experiences** where visual delight is key
