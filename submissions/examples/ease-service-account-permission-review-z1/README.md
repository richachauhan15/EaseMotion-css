# Ease Service Account Permission Review

## Overview

A responsive HTML and CSS dashboard component for reviewing service account permissions, identifying excessive privileges, and monitoring security compliance.

## Features

- Permission review status
- Least-privilege monitoring
- Over-privileged account alerts
- Compliance visibility
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable security dashboard component for identity governance, permission audits, least-privilege enforcement, and enterprise cloud security operations.
