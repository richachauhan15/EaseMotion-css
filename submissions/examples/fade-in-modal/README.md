# CSS Fade-In Modal

A lightweight and accessible Fade-In Modal animation component built using pure HTML and CSS.

## Features

- Pure CSS animation
- No JavaScript dependency
- Smooth fade-in effect
- Responsive design
- Accessible reduced-motion support
- Easy customization using CSS variables


## Usage

Copy the HTML structure from `demo.html` and include the stylesheet.

Example:

```html
<div class="modal">
    <div class="modal-content">
        <h2>Modal Title</h2>
        <p>Modal content here.</p>
    </div>
</div>
