# Ease Application Secret Rotation Tracker

## Overview

A responsive HTML and CSS component for monitoring secret rotation schedules, expiration dates, and security compliance across applications.

## Features

- Secret rotation status
- Expiration monitoring
- Compliance tracking
- Overdue credential alerts
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable security dashboard component for tracking credential rotation, improving compliance, reducing security risks, and supporting enterprise DevSecOps workflows.
