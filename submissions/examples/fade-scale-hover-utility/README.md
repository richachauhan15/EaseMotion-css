# Fade Scale Hover Utility

## Description

Fade Scale Hover Utility is a reusable CSS animation that smoothly enlarges an element while slightly reducing its opacity on hover. It creates a clean and modern interaction suitable for cards, buttons, images, and feature sections.

## Features

- Smooth scale animation
- Subtle fade effect
- Lightweight and reusable
- Pure HTML & CSS
- Easy to customize

## Usage

```html
<div class="fade-scale-hover">
  <h2>Hover Me</h2>
  <p>Smooth fade and scale hover effect.</p>
</div>
```
