\# Zoom-In Pricing Table Component (`ease-pricing-zoom-ak`)



A modern, responsive CSS zoom-in pricing table component built for creative portfolio layouts using \*\*EaseMotion CSS\*\*.



\## Features

\- \*\*Keyframe Zoom Entrance\*\*: Smoothly scales elements into place from an initial reduced scale using a custom cubic-bezier bounce.

\- \*\*Glassmorphic Surface\*\*: Built using native `backdrop-filter: blur()` layers and subtle gradient borders.

\- \*\*Accessibility Ready\*\*: Fully complies with `@media (prefers-reduced-motion: reduce)`.

\- \*\*Zero JavaScript Dependencies\*\*: Powered completely by high-performance native CSS animations.



\## Quick Usage

1\. Place this folder under:

&#x20;  `submissions/examples/pricing-table-zoom-ak/`

2\. Open `demo.html` in any browser to review the entrance effect.
