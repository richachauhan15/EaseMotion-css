# CSS Slide-Up Drawer

## Overview

A responsive pure HTML & CSS slide-up drawer that smoothly appears from the bottom of the screen without JavaScript.

## Features

- Pure HTML & CSS
- Responsive layout
- Bottom slide animation
- Accessible checkbox toggle
- Mobile friendly
- prefers-reduced-motion support
- Lightweight and reusable

## Folder

```
slide-up-drawer/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<input type="checkbox" id="drawer-toggle">

<label for="drawer-toggle" class="drawer-btn">
Open Drawer
</label>

<div class="drawer">
...
</div>
```

## Customization

Modify the following CSS properties:

- Drawer background
- Animation duration
- Border radius
- Overlay opacity
- Button colors
