# Soft Bounce Hover Utility

## Description

Soft Bounce Hover Utility adds a gentle bounce animation when an element is hovered. It creates a smooth, playful interaction suitable for buttons, cards, icons, feature boxes, and other UI components.

## Features

- Smooth bounce animation
- Lightweight and reusable
- Pure HTML & CSS
- Easy to customize
- No JavaScript required

## Usage

```html
<div class="soft-bounce-hover-effect">
  <h2>Hover Me</h2>
  <p>Enjoy a smooth bounce animation.</p>
</div>
```
