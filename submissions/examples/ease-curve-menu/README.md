# 🌊 Curve Menu – Items Follow a Curved Path

> A dynamic, organic menu where items follow a beautiful curved path when they appear, creating a fluid, elegant, and engaging navigation experience.

---

## 📖 Description

The **Curve Menu** is an elegant UI component where menu items follow a graceful curved path when they appear. Each item moves along a sweeping arc, creating a fluid, organic, and visually stunning entrance animation. The items radiate outward from a central toggle button in a gentle curve, mimicking the natural motion of objects following a curved trajectory.

The component features six menu items arranged along a beautiful arc, each with its own unique color gradient. Items have staggered delays, smooth transitions, and tooltip labels on hover. The center toggle button morphs from a plus (+) to a close (×) icon when the menu is open, providing clear visual feedback.

Perfect for creative portfolios, modern interfaces, and any application where you want a fluid, organic, and visually striking menu interaction. The Curve Menu transforms a simple navigation into an elegant, memorable experience.
